**职责链模式：**先捕获一个异常并进行职责内的处理，接着抛出另一个异常，并把原始异常信息保存下来

```java
package jwz;

// 自定义能够接收并保存原始异常的异常类
class SalException extends Exception{
    public SalException(){}
    public SalException(String msg){
        super(msg);
    }
    // 创建一个可以接受Throwable参数的构造器
    public SalException(Throwable t){
        super(t);
    }
}

// 职责链模式[异常链]
public class CalSal{
    public static void main(String[] args) throws SalException{
        try{
            // 业务逻辑
            System.out.println("计算工资");
            System.out.println();
            throw new SalException("异常:职责链模式");
        }catch(Exception e){
            // 记录原始异常，留给管理员
            e.printStackTrace();
            System.out.println();
            // 1. 抛出业务异常，即用户提示
            throw new SalException("系统出现未知异常");
            // 2. e就是原始异常,由main抛出，JVM打印异常跟踪栈信息并中止程序
            throw new SalException(e);
        }
    }
}
```

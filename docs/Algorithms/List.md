# 线性表

## 单链表

### 时间复杂度分析

1. 添加
    - add O(n/2)=O(n)
    - addLast O(n)
    - addFirst O(1)
2. 查找
    - get O(n/2)=O(n)
    - getLast O(n)
    - getFirst O(1)
    - -
    - contains O(n/2)=O(n)
    - indexOf O(n/2)=O(n)
3. 修改
    - set O(n/2)=O(n)
4. 删除
    - remove O(n/2)=O(n)
    - removeLast O(n)
    - removeFirst O(1)
    - -
    - removeElement O(n/2)=O(n)

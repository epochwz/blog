# 直接插入排序

**直接插入排序\(Straight Insertion Sort\)：**将无序表中的一个记录插入到有序表中，从而得到一个新的、记录数+1的有序表，直至无序表为空

* 记录数比较少、记录基本有序时效率极高
* 基本有序：小的关键字基本在前，大的关键字基本在后

# 希尔排序

**希尔排序\(Shell Sort\)**

* 将**待排序列**中相距"**增量n**"的记录组成一个**子序列**
* 对**n个子序列**分别进行直接插入排序
* **缩小增量n**直至1，从而使待排序列**基本有序**
* n=1时，即是对**整个待排序列**进行直接插入排序

由于增量是跳跃式移动，希尔排序是不稳定排序

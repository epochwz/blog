# 链式存储结构

**线性表的链式存储结构**：用一组任意的\[连续或不连续\]的存储单元存储线性表的数据元素,简称**链表**

**结点**：链表的基本组成单元，代表数据元素

* 数据域：存储数据元素的数据信息
* 指针域：存储直接后继元素的_存储单元地址_,称为后继指针,简称_指针_

**头结点**：第1个链表结点前的附加结点

* 数据域：存储链表的附加信息
* 指针域：存储链表第一个有效结点的指针

**头指针**：链表中第1个结点的存储地址，通常用其代表链表

**分类**

* **单链表**：结点中只含一个指针域的链表

* **循环链表\(Circular LinkedList\)**：将单链表的终结点的指针域由空指针改成指向头结点,形成头尾相接的单循环链表

  * 使用尾指针指向终结点,可以快速访问最前/后1个元素

* **双向链表\(Double LinkedList\)**：具有前驱结点指针域+后继结点指针域的单链表

## 单链表

**单链表的时间复杂度**

* 存/取数据元素的时间复杂度：O\(n\)
* 只需一次移动指针到插入位置后\[O\(n\)\],插入/删除数据元素的时间复杂度：O\(1\)

**单链表的C语言描述**

```
// 单链表的结点定义
typedef struct Node{
    // 数据域：存储数据信息
    ElemType data;
    // 指针域：存储后继结点的地址[指针]
    struct Node *next;
}Node;
// 结点指针，作为头指针代表整个链表
typedef struct Node *LinkList;
```

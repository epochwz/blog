# Linux语言环境

**系统语系变量**

```
# 查看系统当前语系
$ echo $LANG
$ locale
    LANG    定义系统主语系
    LC_ALL  定义系统整体语系

# 查看系统支持的所有语系
$ locale -a

# 查看系统默认语系
$ cat /etc/sysconfig/il8n    # CentOS
$ cat /etc/default/locale    # Ubuntu
```

**Linux中文支持**

```
正确安装中文字体和中文语系

图形界面可以正确支持中文显示
第三方远程工具可以支持中文显示
字符界面需要第三方插件支持
```

**Linux切换系统语言**

```
# 配置系统语言环境
# CentOS
$ vim /etc/default/locale
# Ubuntu
$ cat /etc/sysconfig/il8n
    # 中文支持
    LANG="zh_CN.UTF-8"
    LANGUAGE="zh_CN:zh"

    # 英文支持
    LANG="en_US.UTF-8"
    LANGUAGE="en_US:en"

# 安装语言支持
# System settings->Language Support->安装中文语言
```

# 流程控制

**单分支if条件判断式**

```
if [ 条件判断式 ];then
    程序
fi
或
if [ 条件判断式 ]
    then
        程序
fi
```

**双分支if条件判断式**

```
if [ 条件判断式 ]
    then
        程序1
    else
        程序2
fi
```

**多分支if条件判断式**

```
if [ 条件判断式 ]
    then
        程序1
elif [ 条件判断式 ]
    then
        程序2
else
        程序3
fi
```

**多分支case条件判断式**

```
case "$varName" in
    "var1")
        程序1
        ;;
    "var2")
        程序2
        ;;
    *)
        程序3
        ;;
esac
```

**for循环**

```
for varName in var1 var2 var3
    do
        程序
    done

for (( i=0;i<100;i++ ))
    do
        程序
    done
```

**while循环**

```
while [条件判断式]
    do
        程序
    done
```

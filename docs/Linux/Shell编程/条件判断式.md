# 条件判断式

**判断格式**

```
test [Options] /path/file
[ Options /path/file ]

[Options]
按文件类型判断
按文件权限判断
两个文件的比较
两个整数的比较
字符串的比较
```

**按文件类型判断**

```
-b File    判断文件是否存在 && 是否为块设备文件
-c File    判断文件是否存在 && 是否为字符设备文件
-d File    判断文件是否存在 && 是否为目录文件
-e File    判断文件是否存在
-s File    判断文件是否存在 && 是否非空
-f File    判断文件是否存在 && 是否为普通文件
-L File    判断文件是否存在 && 是否为符号链接文件
-p File    判断文件是否存在 && 是否为管道文件
-S File    判断文件是否存在 && 是否为套接字文件
```

**按文件权限判断**

```
                         -r    -w    -x    -u    -g    -k
判断文件是否存在 && 是否拥有 读/写/执行/SUID/SGID/SBit 权限
```

**两个文件的比较**

```
File1 -nt File2    判断File1的修改时间是否比File2新
File1 -ot File2    判断File1的修改时间是否比File2旧
File1 -ef File2    判断File1和File2的iNode是否一致(判断是否为同一个文件)(判断硬链接)
```

**两个整数的比较**

```
num1 -eq num2    判断num1和num2是否相等
num1 -ne num2    判断num1和num2是否不相等
num1 -gt num2    判断num1是否大于num2
num1 -lt num2    判断num1是否小于num2
num1 -ge num2    判断num1是否大于等于num2
num1 -le num2    判断num1是否小于等于num2
```

**字符串的判断**

```
-z str          判断字符串是否为空
-n str          判断字符串是否非空
str1 == str2    判断str1是否等于str2
str1 != str2    判断str1是否不等于str2
```

**多重条件判断**

```
judge1 -a judge2    逻辑与
judge1 -o judge2    逻辑或
!judge              逻辑非
```

# 常用目录

```
/            # 根目录
/boot        # 启动目录--保存系统启动的相关文件
/etc         # 配置文件
/dev         # 设备文件
/lib         # 系统库目录
/var         # 系统相关文档内容

/mnt         # 系统挂载目录(cdROM挂载光盘;usb挂载U盘)
/media       # (多媒体)挂载目录
/misc        # 挂载目录

/tmp         # 系统临时目录
/proc        # 该目录的数据直接写入内存，不能直接操作
/sys         # 该目录的数据直接写入内存，不能直接操作

/home        # 普通用户主目录
/root        # root用户主目录

/bin         # 保存系统命令(普通用户可以读取的命令)
/sbin        # 保存系统命令(超级用户可以读取的命令)

/usr         # 系统软件资源目录
/usr/lib     # 系统软件库目录
/usr/bin     # 保存软件命令(普通用户可以读取的命令)
/usr/sbin    # 保存软件命令(超级用户可以读取的命令)
```

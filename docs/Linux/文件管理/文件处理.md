# 文件处理

**绝对路径：**参照根目录，从根目录开始查找指定路径

**相对路径：**参照当前目录，从当前目录开始查找指定路径

**建立目录**

```
$ mkdir [Options] [dirName]

[Options]
-p    # 递归创建
```

**切换所在目录**

```
$ cd [dirName]

[快捷操作]
$ cd ~    # 进入当前用户的主目录
$ cd      # 进入当前用户的主目录
$ cd -    # 进入上次目录
$ cd ..   # 进入上一级目录
```

**删除文件或目录**

```
$ rm [Options] [File/dirName]

[Options]
-r            # 删除目录
-f(force)     # 强制删除(不询问是否删除)
```

**复制文件或目录**

```
$ cp [Options] [srcFile/srcDir] [desFile/desDir]

[Options]
-r    # 复制目录
-p    # 连带复制文件属性
-d    # 连带复制链接属性
-a    # 等同于-pdr
```

**剪切/重命名文件或目录**

```
$ mv [srcFile/srcDir] [desFile/desDir]
```

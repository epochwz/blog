# 源代码安装包

```
# 下载源码包softName.tar.gz


# 解压缩源码包
$ tar -zxvf softName.tar.gz -C /desDir/softName


# 进入解压目录
$ cd /desDir/softName


# 编译配置与检查
  1. 自定义软件功能选项
  2. 检测系统环境是否符合安装要求
  3. 将功能选项和检测信息写入Makefile文件,用于后续编译
$ ./configure [-options]
# 查看./configure的使用帮助
$ ./configure --help
# 指定安装路径，约定/usr/local/
$ ./configure --prefix=setupPath


# 编译安装
$ make && make install
# 编译失败时应清除编译
$ make clean


# 启动程序
$ /usr/local/xxx/bin/xxx start
```

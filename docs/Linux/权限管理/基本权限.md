# 基本权限

**查看文件权限**

```
# 查看文件权限(详细信息)
$ ll file/dir
文件类型+文件权限   所有者 所属组 文件大小 最后修改时间     文件名称
-rw-r--r--.      root  root  1270    1月 14 18：18   test.txt

# 文件类型：-文件    d目录    |软链接

# 文件权限
rw-  # u(user)     所有者权限
r--  # g(group)    所属组权限
r--  # o(others)   其他人权限
.    # ACL权限
```

**修改文件权限**

```
# 修改文件权限
$ chmod [Options] 权限模式 文件名/目录

[Options]
-R    # 递归

[权限模式]
r(read)    # 4    读权限
w(write)   # 2    写权限
x          # 1    执行权限

# 使用权限模式表示的文件权限
[u g o a][+ - =][r w x]
[777]

# 示例
# 赋予   其他用户 写权限
$ chmod  o+w    fileName

# 赋予全部用户全部权限
$ chmod a=rwx    fileName
$ chmod user 777 fileName
```

**权限的作用**

```
# 权限对文件的作用
r    读取文件内容(cat more head tail)
w    编辑、新增、修改文件内容(vim echo)，但不能删除文件
x    可执行

# 权限对目录的作用
r    查询目录下的文件名(ls)
w    修改目录结构:新建/删除/重命名/剪切文件和目录(touch rm mv cp)
x    可以进入目录(cd)
```

**修改文件所有者**

```
$ chown user file/dir
$ chgrp group file/dir
$ chown user.gropu file/dir
```

**文件默认权限**

```
# 查看默认权限
$ umask
0022
0      文件特殊权限
022    文件默认权限

umask定义的是系统默认丢弃的权限

# 文件的默认权限
文件默认不能建立为执行文件，必须手工赋予执行权限
文件默认权限最大为666
文件默认权限=666-umask值(换算成字母后相减)

# 目录的默认权限
目录默认权限最大值为777

# 临时修改
$ umask xxxx
# 永久修改
$ vim /etc/profile
```

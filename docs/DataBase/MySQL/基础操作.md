**创建数据库**

```sql
CREATE {DATABASE | SCHEMA} {IF NOT EXISTS} dbName [DEFAULT] CHARACTER SET charset_name
```

**删除数据库**

```sql
DROP {DATABASE | SCHEMA} {IF EXISTS} dbName
```

**修改数据库编码方式**

```sql
ALTER DATABASE dbName CHARACTER SET=charset
```

**查看数据库库列表**

```sql
SHOW {DATABASE | SCHEMAS} [LIKE 'pattern' | WHERE expr]
```

**查看错误信息**

```sql
SHOW WARNINGS
```

**查看数据库的创建指令**

```sql
SHOW CREATE DATABASE dbName
```

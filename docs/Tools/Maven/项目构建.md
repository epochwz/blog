# 项目构建

**坐标和仓库**

```bash
坐标(项目的唯一标识)
    <!-- 项目包名 -->
    <groupId>组织名(网址反写).项目名</groupId>
    <!-- 项目名-模块名 -->
    <artifactId>项目名-模块名</artifactId>
    <!-- 快照版本 -->
    <version>1.0-SNAPSHOT</version>
仓库(项目的存储位置)
    本地仓库
    远程仓库(Maven中央仓库)
    镜像仓库
```

**完整的项目构建过程**

```bash
清理->编译->测试->打包->集成测试->验证->部署
```

**Maven生命周期**

```bash
clean    清理项目
    pre-clean    执行清理前的动作
    clean        清理上一次构建生成的所有文件
    post-clean   执行清理后的文件
default  构建项目
    compile      编译
    test         测试
    package      打包
    install      安装JAR项目到Maven本地仓库
site     生成项目站点
    pre-site    生成项目站点前执行的动作
    site        生成项目的站点文档
    post-site   生成项目站点后执行的动作
    site-deploy 发布生成的站点到服务器上
```
